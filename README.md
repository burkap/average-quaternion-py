# Averaging Quaternions
Python implementation of the algorithm proposed in the following paper (by F. Landis Markley, Yang Cheng, John L. Crassidis, Yaakov Oshman):

https://ntrs.nasa.gov/api/citations/20070017872/downloads/20070017872.pdf

 
**! The code is based on Tolga Birdal's MATLAB implementation.**
